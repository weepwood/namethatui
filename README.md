# Name That UI · 中文版

[NameThatUI](https://namethatui.com/) 首页的中文学习复刻版本。项目使用原生 HTML、CSS 与 JavaScript 独立实现，保留视觉词典的核心信息架构，并将页面文案与 62 个 UI 术语翻译为中文。

## 功能

- 62 个 Web / macOS UI 术语卡片
- 响应式瀑布流布局
- 中文、英文与代码关键词搜索
- `⌘/Ctrl + K` 命令面板
- 平台筛选、热门/最新排序、随机推荐
- 术语详情与编程 Agent 提示词复制
- 双击术语显示通俗中文解释
- GitHub Pages 自动部署工作流

## 本地运行

```bash
npm install
npm run dev
```

也可以直接使用任意静态文件服务器打开项目根目录。

## 构建

```bash
npm run build
npm run preview
```
